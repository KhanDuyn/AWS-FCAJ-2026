# AWS Study Group Meetup

## Event information

- **Event name:** AWS Study Group Meetup
- **Date:** June 6, 2026
- **Time:** 09:00 AM – 12:00 PM
- **Location:** Bitexco Financial Tower, Ho Chi Minh City
- **Role:** Participant

This knowledge-sharing event covered AWS, Cloud, DevOps, AI/ML, Docker, security, teamwork, and career orientation. Speakers combined technical demonstrations with practical experience from learning, working, and building cloud-based projects.

## Event objectives

The event aimed to broaden participants' knowledge of AWS, Docker, WebSocket, GraphRAG, Machine Learning, and DevOps. It also provided perspectives on career development, teamwork, and applying cloud knowledge to real-world problems.

## Main sessions

### From IT Helpdesk to Senior Sysadmin – Vinh Tran

Vinh Tran shared his journey from IT Helpdesk to Senior Sysadmin and the first steps toward Cloud/DevOps. Key topics were troubleshooting, user communication, Linux and networking, hands-on labs, system administration, work automation, and interview experience at Central Retail Group.

### AWS Neptune for Building a Graph Knowledge Base for GraphRAG – Viet Phat

The session explained how GraphRAG improves retrieval by storing relationships between entities. It covered the limitations of traditional RAG, implementation with Amazon Bedrock Knowledge Bases and Amazon Neptune Analytics, and a custom approach using LlamaIndex, Amazon Neptune, graph traversal, and Cypher queries.

### The Art of Effective Teamwork – Truong Huy Phuoc

This session emphasised coordination, task distribution, and communication within a team. It also introduced Trello, ClickUp, Google Workspace, Slack, and Discord as tools for managing tasks, sharing information, and tracking progress.

### Docker – A Containerization Technology – Bao Huynh

Bao Huynh introduced Docker and containerization, including the differences from virtualization; Docker images, containers, Dockerfiles, and commands; and use cases in CI/CD, microservices, development/testing environments, and cloud-native applications.

### Combining AWS WAF with Machine Learning for Cyber Attack Detection – Le Hoang Gia Dai

The speaker presented an intrusion-detection solution that combines AWS WAF with Machine Learning. The content covered the limitations of rule- and signature-based security, NIDS, the CSE-CIC-IDS2018 dataset, model training and evaluation, AWS deployment architecture, and a real-time attack-monitoring dashboard.

### Multiplayer in the Cloud: Connecting Godot Clients with AWS WebSockets – Nguyen Quoc Bao

This presentation demonstrated a multiplayer game system using Godot, API Gateway WebSocket, Lambda, and DynamoDB. It covered WebSocket client connections, route keys, player-state storage, Lambda matchmaking, message exchange, and challenges such as stale connections, DynamoDB Scan costs, and stateless Lambda limitations.

## Results and values gained

- Gained practical understanding of AWS, DevOps, Docker, Cloud Security, AI/ML, GraphRAG, and WebSocket.
- Learned career-development perspectives from IT Helpdesk and Sysadmin to Cloud/DevOps.
- Understood how Docker supports consistent application packaging for CI/CD, microservices, and cloud-native applications.
- Learned how Amazon Neptune and Amazon Bedrock can support GraphRAG applications.
- Understood how AWS WAF and Machine Learning can improve detection of abnormal cyberattack behaviour.
- Learned the fundamentals of building a multiplayer system with Godot, WebSocket, Lambda, and DynamoDB.
- Strengthened awareness of teamwork, communication, task allocation, and team-management tools.
