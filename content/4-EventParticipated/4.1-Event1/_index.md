# FCAJ Community Day

## Event information

- **Event name:** FCAJ Community Day
- **Date:** June 27, 2026
- **Time:** 09:00 AM – 12:00 PM
- **Location:** Bitexco Financial Tower, Ho Chi Minh City
- **Role:** Participant

FCAJ Community Day was a community-sharing session organized by AWS Study Group. It focused on AWS Cloud, AI agents, DevOps automation, and cloud architecture, while connecting students and young technology practitioners with the First Cloud Journey/AWS Study Group community.

## Event objectives

The event helped participants learn practical approaches from industry speakers, understand how AWS can be applied to AI and DevOps use cases, and develop an architecture-oriented problem-solving mindset.

## Main sessions

### AgenticOps for Your Cloud – CloudThinker – Steve Tran

Steve Tran introduced the **CloudThinker** mindset: using cloud is not only about knowing AWS services, but also about analysing problems, selecting suitable solutions, designing systems, and applying cloud in real situations. The session highlighted cloud fundamentals, architectural thinking, and continuous learning as foundations for growth in technology.

### AWS DevOps Agent – Bao and Nguyen Nguyen

This session demonstrated how AI can support DevOps teams in monitoring, detecting, and handling system incidents. The incident-response demo showed how a DevOps Agent can analyse issues, suggest remediation directions, and reduce response time when cloud infrastructure encounters problems.

### AI-Powered Productivity – Truong and Minh Anh

The speakers discussed HR challenges in the AI era, including recruitment, candidate management, and workflow optimisation. A demonstration of Amazon Q for recruitment management illustrated how AI can search for information, process candidate data, and improve business operations.

### Building Secure Private MCP for Amazon Q – Toan Nguyen and Nghi

This session explored using Amazon Q in enterprise environments. It explained how a secure private MCP connection can help Amazon Q query internal information, automate tasks, and connect effectively with internal data to improve productivity and decision-making.

## Results and values gained

- Better understood how AWS and AI can solve business problems in DevOps, HR, and enterprise information management.
- Learned the CloudThinker mindset: analyse the problem, choose an appropriate solution, and design scalable systems rather than only operating cloud services.
- Understood how AI-assisted DevOps can help detect, analyse, and respond to infrastructure incidents.
- Learned practical applications of Amazon Q for information retrieval, data management, recruitment, and workflow automation.
- Gained perspectives to apply to AWS learning, teamwork, and future project development.
